# Requisitos funcionales

- El sistema permite autenticarse con email y contrasena.
- El candidato puede consultar y actualizar su perfil.
- El candidato puede cargar un CV para extraer informacion inicial.
- El sistema genera recomendaciones de vacantes ordenadas por score.
- El sistema explica el matching mostrando skills coincidentes y faltantes.
- El candidato puede registrar una postulacion desde la interfaz.
